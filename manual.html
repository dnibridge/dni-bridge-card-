✅ 앱 설치 없이 인터넷만 되면 어디서든 접속 가능합니다. PC, 스마트폰, 태블릿 모두 OK!

사용내역 입력하기
카드를 사용할 때마다 기록해요
1

2
카드 선택

사용 금액 입력
금액을 입력하면 사용 후 남는 잔액이 실시간으로 보입니다. 잔액보다 많은 금액을 입력하면 자동으로 오류 알림이 뜹니다.
4
카테고리 선택 → 메모 자동 추천
카테고리를 선택하면 관련 메모가 버튼으로 나타납니다. 클릭하면 자동 입력되고, 직접 타이핑도 가능합니다.
팀 점심 고객 점심 회의 식사 회식
← 식대 선택 시 이런 버튼이 나타나요
5
저장 클릭 → 잔액 자동 차감 완료!
🚫 잔액 초과 오류 — 카드 잔액보다 많은 금액을 입력하면 저장이 안 되고 오류 팝업이 뜹니다. 카드를 먼저 충전하거나 다른 카드를 선택하세요.
4
사용내역 수정 / 삭제
잘못 입력했을 때 수정하는 방법
1
사용내역 목록에서 해당 행 찾기
화면 하단 사용내역 표에서 수정할 내역을 찾으세요. 부서별 탭으로 필터링하면 빠르게 찾을 수 있어요.
2
맨 오른쪽 ✏️ 버튼(수정) 또는 🗑 버튼(삭제) 클릭
✏️ 수정: 금액, 카테고리, 메모, 상태 모두 변경 가능
🗑 삭제: 삭제하면 카드 잔액도 자동으로 복원됩니다
✅ 수정 시 잔액도 자동으로 다시 계산돼요. 따로 잔액을 건드릴 필요 없어요.
5

상황방법
이름/부서 변경이름, 부서 칸 수정 후 저장 (금액 빈칸 OK)

💡 예: 현재 잔액이 50,000원인데 300,000원을 충전했다면 → 충전 금액 칸에 300,000 입력 → 잔액이 350,000원이 됩니다.

8
자주 묻는 질문
Q. 잔액 부족 오류가 뜨면 어떻게 하나요?
카드 잔액보다 큰 금액을 입력하면 오류 팝업이 뜹니다. ① 금액을 줄이거나 ② 카드 수정(✏️)에서 추가 충전 후 다시 입력하세요.
Q. 새로고침(F5)하면 데이터가 사라지나요?
아니요! 데이터는 브라우저에 자동 저장됩니다. 새로고침하거나 창을 닫아도 데이터가 그대로 남아 있어요.
Q. 스마트폰에서도 쓸 수 있나요?
네! 스마트폰 브라우저에서 주소를 입력하면 바로 사용 가능합니다. 화면이 좁으면 가로 방향으로 돌려서 보세요.
Q. 다른 컴퓨터에서 데이터를 보려면?
현재 버전은 각 기기의 브라우저에 저장되는 방식이라, 다른 기기에서는 별도로 입력해야 합니다. 여러 기기에서 공유하려면 CSV로 저장해서 공유하거나, 관리자에게 공유 기능 추가를 요청하세요.
Q. 실수로 초기화 버튼을 눌렀어요!
초기화 전에 확인 팝업이 뜹니다. "확인"을 누른 경우 데이터가 삭제됩니다. 정기적으로 CSV를 저장해두시면 복구에 도움이 됩니다.
Q. 카테고리를 추가하고 싶어요
카테고리 추가는 시스템 관리자에게 요청해주세요. 현재 제공 카테고리: 식대, 교통/출장, 접대비, 야식비, 사무용품, 연료, 기타
