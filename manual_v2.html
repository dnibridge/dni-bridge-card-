법인에서 직원들에게 충전식 카드(B카드)를 줄 때, 누가 얼마를 썼는지, 얼마가 남았는지를 한눈에 볼 수 있게 해주는 시스템이에요.
마치 통장 잔액처럼, 카드에 돈을 넣고 — 쓸 때마다 기록하고 — 잔액이 줄어드는 걸 실시간으로 확인할 수 있어요.

💳 카드 잔액 관리
누구 카드에 얼마가 남았는지 색깔로 바로 확인. 빨간색이면 충전 필요!
📋 사용내역 기록
언제·누가·얼마를·어디에 썼는지 카테고리별로 깔끔하게 기록
📊 부서별 통계
영업부·관리팀 등 부서별로 얼마나 썼는지 자동 집계·차트
📥 구글시트 연동
직원이 폼으로 입력 → 시트 자동저장 → 대시보드로 가져오기
1
처음 시작하기 — 접속 방법
앱 설치 없이 인터넷 브라우저로 바로!
1
인터넷 브라우저를 열어요 (크롬, 사파리, 엣지 모두 가능해요)
2
주소창에 아래 주소를 입력하고 Enter를 눌러요
dni-bridge-card.vercel.app
3
즐겨찾기(★)에 저장해두면 다음부터 바로 열 수 있어요!
✅ 스마트폰에서도 똑같이 접속할 수 있어요. 홈 화면에 아이콘으로 저장하면 앱처럼 쓸 수 있어요!
2
방법 A — 직접 등록해서 관리하기
관리자가 직접 카드 추가·충전·사용내역을 입력하는 방법

A-1. 카드 추가하기

1
오른쪽 상단 + 카드 추가 버튼을 클릭해요
2
아래 내용을 입력해요
입력 항목설명예시
담당자 이름카드를 사용하는 직원 이름강지원
부서소속 부서명영업부
카드 끝 4자리카드 뒷면 번호 마지막 4자리1234
3
저장 버튼 클릭 → 카드 목록에 바로 나타나요!
💡 카드 추가 시 충전금액은 입력하지 않아요. 충전은 아래 A-2 에서 따로 해요!

A-2. 충전 등록하기

1
오른쪽 상단 ⚡ 충전 등록 버튼 클릭
2
충전할 카드 선택 → 충전 금액 입력 → 메모 입력 (예: 5월 충전)
3
충전 등록 버튼 클릭 → 카드 잔액이 바로 올라가요!
✅ 충전 등록하면 카드 잔액이 자동으로 증가해요. 추가 충전도 같은 방법으로 하면 합산돼요!

A-3. 사용내역 등록하기

1
오른쪽 상단 + 사용내역 등록 버튼 클릭
2
아래 내용 입력
입력 항목설명
카드 선택사용한 직원의 카드 선택 (잔액이 자동 표시돼요)
날짜오늘 날짜 자동 입력 — 이전 날짜 소급 등록도 가능
금액사용 금액 입력 (입력하면 아래에 34,000원 식으로 표시)
카테고리식대/교통·출장/접대비/야식비/사무용품/주유비/기타
메모카테고리 선택 시 추천 버튼 자동 표시 — 클릭하면 자동 입력!
3
저장 클릭 → 잔액 자동 차감!
🚫 잔액보다 많은 금액을 입력하면 "잔액 초과" 팝업이 뜨고 저장이 안 돼요. 먼저 충전해주세요!
⚠️ 이미 등록된 내역과 동일하면 "중복 감지" 팝업이 뜨고 그래도 등록할지 물어봐요.

A-4. 등록된 내역 수정·삭제

1
사용내역 표에서 수정할 행을 찾아요
2
맨 오른쪽 ✏️ 버튼 → 수정 / 🗑 버튼 → 삭제
✅ 수정하거나 삭제하면 카드 잔액도 자동으로 다시 계산돼요!
3
방법 B — 구글폼으로 직원이 등록하기
직원이 직접 폼으로 입력 → 시트에 자동 저장 → 대시보드로 가져오기

B-1. 구글폼 만들기 (처음 1번만)

1
forms.google.com 접속 → + 새 양식 클릭
2
제목: 법인카드 사용내역 등록
3
아래 4개 질문 추가 (+ 버튼으로 추가)
질문유형
담당자 이름을 입력하세요단답형 (필수)
사용 금액 (원, 숫자만)단답형 (필수)
카테고리 선택객관식: 식대/교통·출장/접대비/야식비/사무용품/주유비/기타
메모 (예: 고객 점심)단답형
4
응답 탭 → 초록 스프레드시트 아이콘 클릭 → 구글 시트 연결
✅ 이제 폼에 응답이 들어올 때마다 구글 시트에 자동으로 기록돼요!

B-2. 직원에게 폼 링크 공유하기

1
구글 폼 오른쪽 상단 보내기 버튼 클릭
2
링크 탭 → URL 단축 체크 → 링크 복사
3
직원들에게 카카오톡·문자·이메일로 전송!
💡 직원은 구글 계정이 없어도 돼요. 링크만 열면 바로 입력 가능!

B-3. 시트 데이터를 대시보드로 가져오기

1
구글 시트 열기 → 파일 → 다운로드 → CSV 파일(.csv) 클릭
2
대시보드에서 오른쪽 상단 📥 시트 가져오기 버튼 클릭
3
📁 CSV 파일 전체 탭 → 파일 선택 → 열 자동 감지 확인 → 가져오기 클릭
4
또는 📋 붙여넣기(1건) 탭 → 구글 시트에서 행 Ctrl+C → 입력창에 Ctrl+V → 가져오기
✅ 중복 내역은 자동으로 건너뛰어요. 이미 등록된 건은 다시 들어오지 않아요!
⚠️ 붙여넣기 방법에서 중복 감지되면 가져오기 버튼이 비활성화돼요. 수기 등록 버튼을 사용해주세요.
4
대시보드 화면 설명
각 영역이 무엇을 보여주는지
화면 영역설명
💳 총 카드 수현재 등록된 법인카드 총 개수
💰 총 잔액모든 카드의 충전 잔액 합계
📊 이번 달 사용이번 달 누적 사용금액
🚨 부족 카드잔액 10만원 미만 카드 수 — 빨간색으로 표시
카드별 잔액 현황담당자별 잔액을 막대와 색깔로 표시 (초록: 정상, 노랑: 주의, 빨강: 부족)
부서별 사용 현황부서별 사용금액 합계 막대 차트
📋 사용내역 탭전체 사용내역 목록. 부서 탭으로 필터링 가능
⚡ 충전내역 탭언제 얼마씩 충전했는지 전체 이력
💡 상단 ⚠️ 노란 경고 박스가 뜨면 잔액 부족 카드가 있다는 신호예요. 즉시 충전해주세요!
5
충전내역 관리
언제 얼마를 충전했는지 기록·확인
1
하단 탭에서 ⚡ 충전내역 탭 클릭
2
날짜·담당자·충전금액·메모 전체 이력 확인 가능
3
잘못 입력한 경우 ✏️ 수정 / 🗑 삭제 — 삭제 시 카드 잔액도 자동 조정
✅ 충전 등록 버튼은 상단 ⚡ 버튼과 충전내역 탭 안쪽 버튼 두 곳 모두 사용 가능해요!
6
엑셀(CSV)로 내보내기
정산·보고서 작성에 활용
1
오른쪽 상단 ⬇ CSV 버튼 클릭
2
파일이 자동으로 다운로드돼요 (파일명: DNI_BRIDGE_날짜.csv)
3
다운로드된 파일을 더블클릭 → 엑셀에서 자동으로 열려요
⚠️ 한글이 깨지면: 엑셀에서 파일 열 때 인코딩을 UTF-8로 선택해주세요.
7
자주 묻는 질문
Q. 새로고침(F5)하면 데이터가 사라지나요?
아니요! 데이터는 브라우저에 자동 저장돼요. 창을 닫아도 그대로 남아 있어요.
Q. 잔액 초과 오류가 뜨면?
카드 잔액보다 큰 금액을 입력하면 저장이 안 돼요. ⚡ 충전 등록으로 먼저 충전한 뒤 다시 입력해주세요.
Q. 구글폼 데이터 가져올 때 중복이 걱정돼요
CSV 파일 전체 가져오기는 동일한 날짜+담당자+금액+메모 조합이면 자동으로 건너뛰어요. 붙여넣기 방법은 중복이면 가져오기 버튼이 비활성화되고, 수기 등록을 안내해줘요.
Q. 이전 날짜로 소급 등록할 수 있나요?
네! 사용내역 등록 시 날짜 선택창에서 이전 날짜를 선택하면 돼요. 기본값은 오늘 날짜예요.
Q. 다른 직원도 같이 쓸 수 있나요?
현재 버전은 각 기기별로 데이터가 저장돼요. 여러 명이 공유하려면 구글폼으로 데이터를 모은 뒤 관리자가 CSV로 가져오는 방법 B를 쓰세요.
Q. 실수로 초기화 버튼을 눌렀어요!
초기화 전에 확인 팝업이 뜨고 확인을 눌러야 지워져요. 평소에 CSV 내보내기로 백업해두는 걸 추천해요.
